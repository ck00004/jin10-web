🤖 AI 提供商配置
类型说明：
openai — OpenAI 官方或任何 OpenAI 兼容 API（本地 Ollama、One API、各类代理均可）
claude — Anthropic Claude 官方或兼容接口
gemini — Google Gemini 官方接口
minimax — MiniMax 官方接口（Anthropic 兼容）

远程地址（baseUrl）： 填写 API 的根路径（不含最后的斜杠），留空使用各提供商默认地址。
例如：https://api.openai.com/v1、http://localhost:11434/v1（Ollama）、https://your-proxy.example.com/v1
🌐 服务器设置
修改端口后需要重启服务才能生效
修改监听地址后需要重启服务才能生效，例如：127.0.0.1 仅本地访问
